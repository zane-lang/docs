# Commands for the CLI


| Command    | Description                                          |
|------------|------------------------------------------------------|
| zane run   | Run the project. The binary will not be optimized.   |
| zane build | Build for all available platforms with optimization. |
| zane debug | Show the ast.                                        |
| zane ir    | Show llvm ir.                                        |
